Challenge: Shortcuts

Can you create keyboard shortcuts for your commands? For example:

  • b = Switch to blue pen
  • g = switch to green pen
  • e = switch to eraser
  • c = clear screen

You could even allow the user to change the pen width with the arrow keys!

Save your project
